*srcery-vim*

An occult colorscheme for (n)vim. Perform your dark magick using clearly
defined contrasting colors with a pinch of earthly tint for that lived-in
feel.

==============================================================================
INTRODUCTION						*srcery-introduction*

Created using colors that logically adheres to the 16 color base palette of a
given terminal, while trying to retain its own identity. The colors are
designed to be easy on the eyes yet contrast well with the background for long
sessions using an editor or terminal emulator.

==============================================================================
OPTIONS                                                       *srcery-options*

Srcery includes a few configuration options to customize your experience. To
change any of these you’d put something like this in your `.vimrc` or
`init.lua`:
>vim
  " vimrc:
  let g:srcery_normal_float = 1

>lua
  -- init.lua:
  vim.g.srcery_normal_float = 1

Make sure that you set these variables before assigning `colorscheme`.

BOLD                                                         *srcery-option-bold*

Enables bold text.
>vim
  " Default
  let g:srcery_bold = 1

ITALIC                                                  *srcery-option-italic*

Enables italic text.
>vim
  " Default 
  let g:srcery_italic = 1


UNDERLINE                                            *srcery-option-underline*

Enables underlined text.
>vim
  " Default
  let g:srcery_underline = 1


UNDERCURL                                            *srcery-option-undercurl*

Enables undercurled text.

>vim
  "Default
  let g:srcery_undercurl = 1

STRIKETHROUGH                                    *srcery-option-strikethrough*

Enables strikethrough text.
>vim
  " Default
  let g:srcery_strikethrough = 1


INVERSE                                                *srcery-option-inverse*

Enables inverse highlighting (foreground becomes background, vice
versa) for visual selection, pmenu selection.

Will use shades of gray if its disabled (0)
>vim
  " Default
  let g:srcery_inverse = 1


GUISP_FALLBACK                                  *srcery-option-guisp-fallback*

Use alternate highlighting where colored underline/undercurl is
unsupported. Set to either `'fg'` or `'bg'
>vim
  " Default
  let g:srcery_guisp_fallback = 'NONE'

COLOR                                                           *srcery-color*


You can customize each of Srcery's colors, to customize say the red color:

>vim
  let g:srcery_red = '#FF0000'

>lua
  vim.g.srcery_red = '#FF0000'

You can also separately set a different background color `g:srcery_background`,
instead of the default `g:srcery_black`

Note: Setting this variable to `'NONE'` will make the background inside vim
transparent
